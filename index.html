Tutoriales para Informatica

Linux Debian

Para comenzar en la creacion de servidores web utilizaremos el sistema operativo Debian ya que es uno de los sistemas operativos mas confiables Debian es un sistema operativo libre, desarrollado y mantenido por el proyecto Debian. Una distribución libre de Linux con miles de aplicaciones para cumplir con las necesidades de nuestros usuarios.

DNS

El sistema de nombres de dominio o DNS (Domain Name System) es un conjunto de protocolos y servicios que nació con la finalidad de evitar a las personas usuarias el esfuerzo de tener que recordar una dirección IP. Sin lugar a duda, nos resulta más fácil recordar un nombre, y además la dirección numérica podría cambiar por diferentes razones, sin que por ello cambie el nombre.

Servidor SSH

SSH (o Secure SHell) es el nombre de un protocolo y del programa que lo implementa cuya principal función es el acceso remoto a un servidor por medio de un canal seguro en el que toda la información está cifrada. Además de la conexión a otros dispositivos, SSH permite copiar datos de forma segura (tanto archivos sueltos como simular sesiones FTP cifradas), gestionar claves RSA para no escribir contraseñas al conectar a los dispositivos y pasar los datos de cualquier otra aplicación por un canal seguro tunelizado mediante SSH y también puede redirigir el tráfico del (Sistema de Ventanas X) para poder ejecutar programas gráficos remotamente.

MySQL/MariaDB

MariaDB es un sistema de gestión de bases de datos que está muy relacionado con MySQL. El sistema de gestión de bases de datos MariaDB incorpora las distintas funciones características de MySQL añadiendo algunas mejoras, como la posibilidad de ejecutar consultas complejas y almacenarlas directamente en caché

Crear Usuario en Linux

El concepto de usuario en Linux permite separar entornos de ejecución para diferentes propósitos. Dos personas pueden trabajar simultáneamente en el mismo sistema, teniendo cada uno un usuario diferente, y un directorio personal diferente.

Putty

PuTTY es un emulador de terminal gratuito que admite varios protocolos de red tal como SSH. Esto te permite correr comandos UNIX en tu servidor el cual no está disponible cuando te conectas usando un cliente FTP.
