Levantar un Servidor SSH

El sistema de nombres de dominio o DNS (Domain Name System) es un conjunto de protocolos y servicios que nació con la finalidad de evitar a las personas usuarias el esfuerzo de tener que recordar una dirección IP. Sin lugar a duda, nos resulta más fácil recordar un nombre, y además la dirección numérica podría cambiar por diferentes razones, sin que por ello cambie el nombre.

Pasos para hacerlo

Primero deberemos ingresar en nuestro equipo en donde queremos instalar este servicio, hecho esto, debemos de ser root para hacerlo con el siguiente comando: su - e ingresamos nuestra contraseña de root.

Podemos revisar los puertos que tenemos para ver cuales tenemos disponibles y escoger de esos el que queramos con el siguiente comando: ss -nltup

Una vez hecho lo anterior, ahora procedemos a instalar el paquete con la siguiente instrucción: apt-get install openssh-server, necesario para nuestro servidor SSH

Lo que necesitamos para configurar correctamente son los siguientes puntos.

Deshabilitar iniciar sesión como root

Cambiar el puerto

Crear un usuario

Para empezar a configurar primero nos moveremos de directorio con lo siguiente: cd /etc/ssh/

Dentro podemos revisar el contenido que tiene con ls -l para estar seguros que el documento que vamos a modificar este ahí. Los modificaremos con VI con la siguiente instrucción: vi sshd_config

Dentro debemos modificar 2 linea, Port y PermitRootLogin

En la linea de Port deberemos cambiar el puerto al que tu gustes, de preferencia entre 1025 al 47,000.

Ya terminado la configuración, ahora queda reiniciar el servicio con el siguiente comando: system restart ssh y con ello hemos terminado
